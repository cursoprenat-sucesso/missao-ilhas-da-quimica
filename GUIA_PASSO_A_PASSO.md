# Guia rápido — Missão Ilhas da Química Premium V3

1. Crie um repositório novo chamado `missao-ilhas-da-quimica`.
2. Envie todos os arquivos desta pasta.
3. Ative o GitHub Pages: Settings → Pages → Deploy from a branch → main → / root → Save.
4. O aluno usa o link da raiz do site.
5. O professor usa o link terminado em `professor.html`.
6. Para cadastrar questões, use o painel do professor e baixe `questions.json` e `settings.json`.
7. Envie os JSON atualizados para o GitHub.

A evolução da tartaruga, as patentes e os itens conquistados ficam em `settings.json`.
